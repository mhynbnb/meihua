## How To Use:
运行`main_c.py`

点击`立即起卦`，以当前时间起卦

时间栏可自定义时间，设置好后点击`起卦`即可

选中卦象，点击`查询`可查看周易中关于该卦象的解释

点击`详细`可显示具体卦象解释以及北宋易学家邵雍和台湾国学大儒傅佩荣对卦象的解释（数据来源于网络）

右上角图片会根据卦象变化，方便分析


点击`万物类象`，可查看八卦对应的万物类象

点击`字典`，查询单个字的意思
